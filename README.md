# SIGNAL_LOST

**`https://sbullocks.github.io/signal-lost/`**

---

A self-aware, surrealist single-page web experience. Glitch art meets brutalist design meets experimental digital artifact. The site knows you're there. It reacts to everything.

Pure HTML/CSS/JS — no dependencies, no framework, no build step.

---

## Easter Eggs

There are **7 hidden signals** scattered across the page. Finding all of them unlocks a final transmission.

No hints. That's the point.

---

## What It Does

- Particle field that reacts to your cursor
- Glowing cursor trail
- Eyes that track your mouse
- Live coordinate and session tracking
- Fake terminal with interactive commands (click it, type `help`)
- Typewriter that keeps adding lines
- Drag your mouse across any text to corrupt it
- Ambient audio layer (opt-in)
- Loading bars that never finish
- Right-click is overridden
- Tab switching triggers title changes
- Idle detection with escalating sass
- Random connection drop overlay
- Exit intent detection
- Visit counter (it remembers you)
- 3AM mode for late-night visitors
- Animated favicon
- Parallax scroll drift
- Worst dark mode (triple-click the footer signature)
- Screen reader hidden message
- HTML source comment easter egg
- Downloadable `signal.txt` on completion

---

## Tech

- Single `index.html` — no dependencies
- Web Audio API for ambient static/glitch sounds
- Canvas API for particles and animated favicon
- `localStorage` for visit persistence
- `DeviceMotionEvent` for mobile shake detection
